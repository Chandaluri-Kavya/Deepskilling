# HOL08 — HTTP Client, RxJS & Interceptors

Corresponds to PDF Hands-On 8 (tasks 78–91). The cumulative runnable solution is in [`../HOL10/student-course-portal`](../HOL10/student-course-portal).

Before running, start JSON Server with `npm run api` from the portal project. The app uses HTTP CRUD service methods, `map`, `tap`, `retry`, `catchError`, `switchMap`, plus auth/error/loading interceptors.
