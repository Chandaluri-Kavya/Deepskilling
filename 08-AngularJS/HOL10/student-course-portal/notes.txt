angular.json: Configures CLI builders, assets, styles, and production build budgets.
tsconfig.json: Holds shared TypeScript compiler options.
tsconfig.app.json: TypeScript settings used for the browser application build.
package.json: Declares scripts and project dependencies.
src/main.ts: Browser entry point that bootstraps the root component.
src/app/app.config.ts: Registers standalone application providers such as router, HTTP, and NgRx.
src/app/app.component.ts: Root shell containing the header and router outlet.
src/index.html: Host HTML document into which Angular renders the app.

Production budgets: maximumWarning (500 kB) warns when the initial bundle grows too large; maximumError (1 MB) fails the production build at that threshold.
