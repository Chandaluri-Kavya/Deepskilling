# HOL10 — Unit Testing & Complete Cumulative Project

Corresponds to PDF Hands-On 10 (tasks 101–110).

The complete runnable Angular 20 solution is in [student-course-portal](student-course-portal). This placement follows the PDF's instruction to build one cumulative project, while the `HOL01`–`HOL09` folders document each matching exercise.

When you are ready to run it manually:

1. `cd student-course-portal`
2. `npm install`
3. In one terminal: `npm run api`
4. In a second terminal: `npm start`
5. For tests: `npm test`

Take your screenshots, review the changes, and commit/push manually afterward.
